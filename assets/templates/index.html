GPT-5 登场

English

OpenAI 于今天凌晨在他们的夏季更新直播上发布了 GPT-5，他们的最新模型。

亮点一览:

  • 所有用户都可以在本周开始体验 GPT-5
  • 免费用户可体验与 GPT-5 发送有限的信息和文件，Plus用户限额更高，Pro用户没有限额
  • 开发者可使用的 API 令牌价格与o3相等
  • 开发者可使用的 API 模型为 GPT-5，GPT-5 mini 和 GPT-5 nano
  • GPT 更聪明了，从写邮件到软件开发速度更快，准确率更高
  • GPT 更智能了，不需要用户特别打开开关就知道调用深度思考
  • GPT 更快了，处理日常要求更敏捷，虽然 OpenAI 对此没有定量比较
  • GPT 更自然了，从写作到说话更像人类
  • GPT 更安全了，处理存在潜在危险的请求时会警告并解释为什么无法处理该请求
  • GPT 更个性化了，Plus 版用户可以调节信息气泡颜色，另添加和日历与邮件同步功能
